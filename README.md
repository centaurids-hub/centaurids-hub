## Laboratories

---

### [Lab 1 — Version Control Systems: Git and GitHub](https://github.com/centaurids-hub/Lab1_Git_Cresencio)

> Introduction to version control using Git, a tool used for tracking changes in projects and managing collaboration among developers. It covers basic Git commands, setting up repositories, saving project changes through commits, creating and managing branches, and working with remote repositories using GitHub for collaboration and project sharing.

`Git` · `GitHub` · `Version Control`

---

### [Lab 2 — Python Fundamentals: Computational Logic & Control Flow](https://github.com/centaurids-hub/Lab2_Fundamentals)

> Introduction to foundational Python programming concepts. It covers different data types, ASCII conversion, string manipulation, conditional statements, exception handling, loops, and basic forensic hash calculation techniques used for data verification and integrity checking.

`Python` · `Fundamentals`

---

### [Lab 3 — Functions and Modules](https://github.com/centaurids-hub/Lab3_FunctionsModules_Cresencio)

> Introduction to functions, scope, recursion, and decorators in Python. Covers function definition, LEGB scope, parametric processing, recursion, decorators, lambda functions, and generators.

`Python` · `Functions` · `Recursion`

---

### [Lab 4 — Data Structures](https://github.com/centaurids-hub/Lab4_DataStructures_Cresencio)

> Introduction to fundamental Python data structures. Covers lists, tuples, dictionaries, sets, arrays, deques, default dictionaries, and list comprehensions.

`Python` · `Data Structures`

---

### [Lab 4. — Algorithms and Big O](https://github.com/centaurids-hub/Lab4.2_Algorithms_and_BigO)

> Introduction to algorithm design and complexity analysis using Big O notation. Covers linear and binary search, bubble, selection, and insertion sort, recursion vs. iteration, and performance analysis.

`Python` · `Algorithms` · `Big O Notation`

---

### [Lab 5 — File Handling: Reading, Processing, and Writing Data](https://github.com/centaurids-hub/Lab5_FileHandling)

> Introduction to file operations in Python. Covers reading, writing, appending, copying, renaming, and deleting files, JSON and CSV handling, exception handling, and menu-driven file management.

`Python` · `File I/O` · `CSV`

---

### [Lab 6 — Numerical and Data Analysis with NumPy and Pandas](https://github.com/centaurids-hub/Lab6_Numpy-Pandas)

> Introduction to numerical computing using NumPy and Pandas. Covers array operations, broadcasting, indexing, reshaping, statistical analysis, data grouping, and DataFrame manipulation.

`Jupyter Notebook` · `NumPy` · `Pandas` · `Python`

---

### [Lab 7 — Data Visualization](https://github.com/centaurids-hub/Lab7_Data_Visalization)

> Introduction to data preprocessing and exploratory visualization using Pandas and Matplotlib/Seaborn. Covers handling missing values, data transformation, outlier detection, and creating meaningful plots for data insights.

`Python` · `Pandas` · `Matplotlib` · `Seaborn`

---

### [Lab 8 — Object Oriented Programming (OOP)](https://github.com/centaurids-hub/Lab8_Object_Oriented_Programming-OOP-)

> Introduction to object-oriented programming in Python. Covers classes and objects, encapsulation, inheritance, polymorphism, class and static methods, operator overloading, abstract classes, and method resolution order (MRO).

`Python` · `OOP` · `Polymorphism`

---

### [Lab 9 — Automating Data Collection through Web Scraping](https://github.com/centaurids-hub/Lab9_Automation_Data_Collection)

> Introduction to web scraping and automated data collection in Python. Covers HTTP requests with headers, HTML parsing using BeautifulSoup, extracting data from tables and paginated pages, data cleaning with regular expressions, error handling, and storing scraped data in CSV, JSON, and SQLite formats.

`Python` · `BeautifulSoup` · `Requests`

---

### [Lab 10 — Databases & SQL with Python](https://github.com/centaurids-hub/Lab10_Databases_and_SQL_wth_Python)

> Introduction to database programming using SQLite in Python. Covers database creation and connection, table creation, inserting and managing records with parameterized queries, filtering and sorting data, updating and deleting records, aggregate functions, JOIN operations, exporting query results to CSV, and error handling.

`Python` · `SQL` · `SQLite` · `Pandas`

> ⚠️ **Note:** In the course PDF, this is listed as **Lab 11**, but the GitHub repository is named `Lab10_Databases_and_SQL`. It is ordered here as Lab 10 because databases and SQL are a prerequisite concept — you need to understand data storage before building a backend web application with Flask.

---

### [Lab 11 — Building Web Applications with Flask](https://github.com/centaurids-hub/Lab11_Web_Applications_With_Flask)

> Introduction to web application development using the Flask framework in Python. Covers project setup with virtual environments, routing and URL handling, Jinja2 templating with template inheritance, static file management (CSS and images), form creation and validation using Flask-WTF, database management using Flask-SQLAlchemy, user authentication with password hashing and Flask-Login, session management, flash messages, and error handling.

`HTML` · `CSS` · `Flask` · `Python` · `Jinja2`

> ⚠️ **Note:** In the course PDF, this is listed as **Lab 10**, but the GitHub repository is named `Lab11_Building_Web_Applications_with_Flask`. It is ordered here as Lab 11 because Flask comes after SQL — once you know how to store and query data, you can then use it to power a dynamic web application.

## Final Project — Engineering Data Systems Pipeline

### [EDS_2043_Cresencio](https://github.com/centaurids-hub/EDS_2043_Cresencio)

> The final project integrates all lab skills into a **production-level Python data analytics pipeline** built on a real-world engineering dataset from Kaggle. Each student is assigned a unique engineering pillar and research topic based on their student number and surname. This project has its co-repository named [COMPROG1_FINAL-PROJECT](https://github.com/centaurids-hub/COMPROG1_FINAL-PROJECT) that has the project documentation and analysis through ipynb file.

> 🧪 *Built on concepts from Lab 6 (Numerical & Data Analysis), Lab 7 (Data Visualization), and Lab 8 (Object-Oriented Programming).*

`Python` · `NumPy` · `Pandas` · `Matplotlib` · `Plotly` · `OOP` · `IEEE Paper` · `Kaggle`
